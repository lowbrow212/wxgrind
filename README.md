# ⛅ Wx Decoder

**Free aviation weather practice quiz** — no account, no subscription, works offline.

🌐 **Live site:** [wxdecoder.github.io](https://wxdecoder.github.io/wxdecoder)

Practice reading and decoding:
- **METAR** — surface observations
- **TAF** — terminal aerodrome forecasts  
- **PIREP** — pilot reports
- **Winds Aloft (FD)** — upper winds forecast
- **SIGMET / AIRMET** — significant and advisory weather

### Features
- **Decode mode** — click any segment, explain what it means
- **Encode mode** — given plain English, write the code
- **Full report mode** — translate every segment one by one
- **Obscure Codes** — the weird ones people always forget (BR, PWINO, PRESFR, etc.) with memory aids
- **Drill Flagged** — flag segments you miss, drill them specifically
- **Go / No-Go scenarios** — full weather briefings, you make the call
- **Offline** — pure HTML/JS, no API calls, no internet required after loading

### Accuracy
All codes and descriptions are based on:
- FAA Aeronautical Information Manual (AIM)
- ICAO Annex 3
- FAA Order 7110.10 (Flight Services)
- AC 00-45 (Aviation Weather Services)

### Disclaimer
This tool is for educational practice only. Always obtain an official weather briefing from an authorized source (1800wxbrief.com, ForeFlight, etc.) before flight.
